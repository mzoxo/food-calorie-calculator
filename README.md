# 食材卡路里計算器

快速計算食材卡路里並加總的靜態網頁應用。

## 功能

- **食材搜尋** — 模糊搜尋名稱、別名、品牌、分類
- **數量輸入** — 克數 / 份數模式切換，+/− 微調
- **群組管理** — 早餐、午餐、晚餐、點心，可自訂
- **常用組合** — 儲存組合，一鍵加入
- **營養總覽** — 群組小計、全日總計、三大營養素比例
- **匯出** — 純文字格式，一鍵複製
- **內建資料庫** — 350+ 筆常見食材，開啟即用

## 使用方式

開啟網頁即可直接使用內建食材資料庫。

如需連結自己的 Google Sheet 食材庫，點右上角齒輪圖示，輸入 API URL 和 TOKEN。

## 技術

純前端靜態網頁（HTML + CSS + JavaScript），無框架、無建置步驟。
